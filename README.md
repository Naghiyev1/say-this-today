# Say This Today v2

A tiny installable daily conversation starter app.

## What is new in v2

- Today view: one daily line.
- Browse view: choose categories and browse all 365 lines.
- Favourites view: see all saved lines.
- Copy any line.
- Save or remove favourites from any view.
- Installable on phone via Add to Home Screen.
- Offline support after first visit.

## Deploy on GitHub Pages

1. Create a new GitHub repository or open your existing one.
2. Upload/replace all files and folders in this ZIP.
3. Go to Settings → Pages.
4. Choose Deploy from a branch.
5. Choose `main` and `/root`.
6. Save.

## Important update note

If you already opened v1 on your phone, your browser may cache it. After uploading v2, open the app, refresh it once or twice, or remove and re-add it to your home screen if needed.
